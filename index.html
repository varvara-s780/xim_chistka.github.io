Мастерская чистоты

Курсовой проект по МДК 09.02

Студент: Капшукова Варвара Андреевна, группа ИС1-33

Тема: Разработка Web-сайта салона химчистки

О проекте:

Разработан полнофункциональный веб-сайт для салона химчистки «Мастерская чистоты».

Полный код проекта (PHP + MySQL + база данных) доступен в этом репозитории.

Для запуска PHP-версии требуется локальный веб-сервер с поддержкой PHP (например, Open Server).

Функционал сайта:

  • Регистрация и авторизация пользователей
  • Просмотр услуг и цен
  • Калькулятор стоимости
  • Бонусная программа (10% от заказа)
  • Оформление заказа с доставкой
  • Личный кабинет с историей заказов
  • Система отзывов
  • Адаптивный дизайн

Технологии:

PHP 7, MySQL, HTML5, CSS3, Bootstrap 5, JavaScript
